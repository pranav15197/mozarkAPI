LOGIN

/functional/user/signin POST

This API is used to login a user. It returns a session identifier that is supposed to be used for further authorized API calls.

Request Syntax

        
    {
  "json": {
    "request": {
      "data": {
        "email": "String",
        "password": "String"
      }
    }
  }
}
email User's email mandatory
password User's password mandatory

Response Syntax

        
    {
    "json": {
        "response": {
            "data": {
                "userDetails": {
                    "userId": 12,
                    "email": "pranav.tiwari@toptal.com",
                    "firstName": "Pranav2",
                    "lastName": "Tiwari",
                    "mobile": "91.7738260941",
                    "organizationAdmin": true,
                    "organizationId": 6,
                    "domain": "TOPTAL.COM",
                    "emailValidated": true,
                    "createdTime": "2020-04-10 11:00:39.0",
                    "firstLogin": true
                }
            },
            "session": {
                "sessionId": "CGZAADAAAAAA"
            },
            "statusCode": "0",
            "statusMessage": "Success"
        }
    }
}
userID unique id for the user
email user's email
firstName user's first name
lastName user's last name
mobile user's mobile
organizationAdmin boolean representing if the user is an admin
organizationId unique Id of the organization that the user is a part of
domain email domain of the user
emailValidated boolean representing if the user's email has been validated
createdTime timestamp when the user was registered.
firstLogin boolean representing if the user is logging in for the first time

Error Codes

Status Code Description
0 Success
1009 Email is not registered yet.
1010 Password is incorrect
