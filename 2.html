Login

http://aquamark.mozark.ai:8080/functional/user/signin POST

This API is used to login a user. It returns a session identifier that is supposed to be used for further authorized API calls.

Request Syntax

              {
  "json": {
    "request": {
      "data": {
        "email": "String",
        "password": "String"
      }
    }
  }
}

Request Parameters

The request accepts the following data in JSON format.

email
User's email or mobile number
Type: String
Required: Yes
password
User's password
Type: String
Required: Yes

Request Headers

The following headers must be sent with the request.

content-type
application/json
Required: Yes

Response Syntax

              {
  "json": {
    "response": {
      "data": {
        "userDetails": {
          "userId": 6,
          "email": "swayam@infosoftjoin.in",
          "firstName": "Swayam",
          "mobile": "9126325823",
          "organizationAdmin": false,
          "companyName": "ISJ",
          "jobTitle": "Technical Architect",
          "validated": true,
          "createdTime": "2020-04-28 11:17:13.0",
          "firstLogin": false
        }
      },
      "session": {
        "sessionId": "XDYAADAAAAAA"
      },
      "statusCode": "0",
      "statusMessage": "Success"
    }
  }
}

Response Elements

userID
unique id for the user
Type: Integer
email
user's email
Type: String
firstName
user's first name
Type: String
mobile
user's mobile
Type: String
organizationAdmin
boolean representing if the user is an admin
Type: Boolean
organizationId
unique Id of the organization that the user is a part of
Type: Integer
domain
email domain of the user
Type: String
emailValidated
boolean representing if the user's email has been validated
Type: Boolean
createdTime
timestamp when the user was registered.
Type: String
firstLogin
boolean representing if the user is logging in for the first time
Type: Boolean
