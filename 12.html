LIST APPS

/functional/app POST

List down all Apps for a given project

Request Syntax

              {
  "json": {
    "request": {
      "session": {
      	"sessionId": "String"
      },
      "data": {
      	"projectId": Integer,
      	"disabled": Boolean
      }
    }
  }
}

Request Parameters

The request accepts the following data in JSON format.

sessionId
Session Identifier
Type: String
Required: Yes
projectId
ID of the project for which apps are requested
Type: String
Required: Yes
disabled
If this is set to true, only apps which are not disabled will be returned
Type: String
Required: No

Response Elements

Number of Apps
Type: String
list
List of Apps
Type: String
