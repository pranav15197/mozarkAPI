LIST APPS

/functional/app POST

List down all Apps for a given project

Request Syntax

        
    {
  "json": {
    "request": {
      "session": {
      	"sessionId": "String"
      },
      "data": {
      	"projectId": Integer,
      	"disabled": Boolean
      }
    }
  }
}
sessionId Session Identifier mandatory
projectId ID of the project for which apps are requested mandatory
disabled If this is set to true, only apps which are not disabled will be returned optional

    {
    "json": {
        "response": {
            "data": {
                "count": 1,
                "list": [
                    {
                        "id": 2,
                        "name": "app-debug.apk",
                        "createdTime": "2020-04-16 08:25:03",

                        "package": "com.example.ayulinkdemoprokect",

                    }
                ]
            },
            "session": {
                "sessionId": "YGAAADAAAAAA"
            },

        }
    }
}
