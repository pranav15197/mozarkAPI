Introduction

The Mozark APIs are a set of REST APIs that allow the user to access all the functionalities of the platform including -

  • Access on-field and on-cloud devices
  • Distribute app and test using Automation scripts (Appium, Appium, UIAutomator, Espresso, Calabash, XCTest UI)
  • Access logs, screen shots
  • Access test history and results

The APIs use a Token based authentication. This token needs to pe passed in the request body. The request and response bodies across all the APIs follow the following structure

Request

              {
  "json": {
    "request": {
      "session": {
      	"sessionId": ""
      },
      "data": {
      	...
      }
    }
  }
}

Response Syntax

              {
  "json": {
    "response": {
      "data": {
        ...
      },
      "session": {
        ...
      },
      "statusCode": "0",
      "statusMessage": "Success"
    }
  }
}
