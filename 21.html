Get details for configured jira account to report bug

/functional/jira/data/{{session_id}} GET

This API returns the details (list of projects, users etc.) of the JIRA account that the user has linked with the platform. Ther is no JSON body required for this API, the user's session id is to be passed in the URL

Request Syntax

              -

Request Parameters

The request accepts the following data in JSON format.

sessionId
Session Identifier
Type: String
Required: Yes
