VERIFY API

/functional/user/signup/verifyemail POST

This API is used to validate a newly registered User's email. A verification code is sent to the user's email post-registration which needs to be passed in this API

Request Syntax

        
    {
  "json": {
    "request": {
      "data": {
        "email": "String",
        "verificationCode": "String"
      }
    }
  }
}
email User's email optional
verificationCode The verification code sent to the user's email optional

Response Syntax

        
    {
    "json": {
        "response": {
            "statusCode": "0",
            "statusMessage": "Email verified successfully"
        }
    }
}

Error Codes

Status Code Description
1007 Email Address is already validated
1008 The verification code and email combination is incorrect
0 success
