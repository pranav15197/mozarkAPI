LIST TRAYS

/functional/tray/list POST

A Tray is a collection of devices to be used for testing. This API returns a list of trays created by the user

Request Syntax

        
    {
  "json": {
    "request": {
      "session": {
      	"sessionId": "String"
      }
    }
  }
}
sessionId Session Identifier mandatory
