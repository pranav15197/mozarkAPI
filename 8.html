http://aquamark.mozark.ai:8080/functional/tray/list POST

A Tray is a collection of devices to be used for testing. This API returns a list of trays created by the user

Request Syntax

              {
  "json": {
    "request": {
      "session": {
      	"sessionId": "String"
      }
    }
  }
}

Request Parameters

The request accepts the following data in JSON format.

sessionId
Session Identifier
Type: String
Required: Yes

Request Headers

The following headers must be sent with the request.

content-type
application/json
Required: Yes
