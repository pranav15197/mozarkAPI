/functional/job/details POST

This API is used to get the details for a particular job. Each job corresponds to a test script running on 1 device for one build of the application. The details contain the status of each test case as well as screenshots of the devices taken during the tests. Please note that these results will be available only when the job is completed.

Request Syntax

              {
  "json": {
    "request": {
      "session": {
        "sessionId": "String"
      },
      "data": {
        "jobId": Integer
      }
    }
  }
}

Request Parameters

The request accepts the following data in JSON format.

sessionId
Session Identifier
Type: String
Required: Yes
jobId
ID of the job
Type: String
Required: Yes
