CREATE PROJECT

/functional/project POST

This API is used to create a new project for the logged in user

Request Syntax

        
    {
  "json": {
    "request": {
      "session": {
      	"sessionId": "String"
      },
      "data": {
      	"name": "String",
      	"description": "String"
      }
    }
  }
}
sessionId session identifier for the logged-in user mandatory
name Name for the project. This needs to be unique for the organization mandatory
description A text describing the project mandatory

Response Syntax

        
    {
    "json": {
        "response": {
            "statusCode": "0",
            "statusMessage": "Success"
        }
    }
}

Error Codes

Status Code Description
1012 A project with the provided name has already been added.
