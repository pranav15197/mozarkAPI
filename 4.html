MOZARK API DOCUMENTATION

Create Project

/functional/project POST

This API is used to create a new project for the logged in user

Request Syntax

              {
  "json": {
    "request": {
      "session": {
      	"sessionId": "String"
      },
      "data": {
      	"name": "String",
      	"description": "String"
      }
    }
  }
}

Request Parameters

The request accepts the following data in JSON format.

sessionId
session identifier for the logged-in user
Type: String
Required: Yes
name
Name for the project. This needs to be unique for the organization
Type: String
Required: Yes
description
A text describing the project
Type: String
Required: Yes

Response Syntax

              {
    "json": {
        "response": {
            "statusCode": "0",
            "statusMessage": "Success"
        }
    }
}
