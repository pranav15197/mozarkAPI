Upload Script

/functional/scriptupload?request= POST

This API is used to upload a script that will be used for automated testing for a project. This script has to be a zip file. This is supposed to be a multipart request. The JSON request body is to be sent through the URL parameter named 'request', and the script zip file has to be sent as a form-data field named 'file'

Request Syntax

              {
  "json": {
    "request": {
      "session": {
        "sessionId": "String"
      },
      "data": {
        "description": "String",
        "language": "String",
        "version": "String",
        "title": "String",
        "projectId":Integer
      }
    }
  }
}

Request Parameters

The request accepts the following data in JSON format.

description
A text describing the script
Type: String
Required: Yes
language
Programming Language used by the script
Type: String
Required: Yes
version
A version of the language used by the script
Type: String
Required: Yes
projectId
ID of the project the script should be added to
Type: String
Required: Yes
title
A title for the script
Type: String
Required: Yes
file
Form data field to be used for the zip script file
Type: String
Required: Yes

Response Syntax

              {
  "json": {
    "response": {
      "data": {
        "scriptId": 1
      },
      "session": {
        "sessionId": "ZBRAAXAAAAAA"
      },
      "statusCode": "0",
      "statusMessage": "Success"
    }
  }
}
