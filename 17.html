/functional/job/list POST

List down the jobs started for a given tray ID, along with their status. One job corresponds to one device in the tray.

Request Syntax

              {
  "json": {
    "request": {
      "session": {
      	"sessionId": "{{session_id}}"
      },
      "data": {
      	"trayId": 3
      }
    }
  }
}

Request Parameters

The request accepts the following data in JSON format.

sessionId
Session Identifier
Type: String
Required: Yes
trayId
ID of the tray for which jobs re required
Type: String
Required: Yes
