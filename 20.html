Configure JIRA account

/functional/jira/configure PUT

This API is used to configure the user's JIRA account with the platform so that bugs can be raised against the test cases.

Request Syntax

              {
    "json": {
        "request": {
            "session": {
                "sessionId": "String"
            },
            "data": {
                "email": "String",
                "token": "String",
                "url": "String"
            }
        }
    }
}

Request Parameters

The request accepts the following data in JSON format.

email
Email for the JIRA user
Type: String
Required: Yes
token
Auth Token for the JIRA user
Type: String
Required: Yes
url
URL for your JIRA project
Type: String
Required: Yes

Response Syntax

              {
  "json": {
    "response": {
      "session": {
        "sessionId": "PY2AAXAAAAAA"
      },
      "statusCode": "0",
      "statusMessage": "Success"
    }
  }
}
