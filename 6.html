UPDATE PROJECT

/functional/project PUT

API used for updating a project's details. This API can be used to delete a project as well.

Request Syntax

        
    {
    "json": {
        "request": {
            "session": {
                "sessionId": "String"
            },
            "data": {
                "id": Integer,
                "description": "String",
                "disabled": Boolean,
                "dashboard": Boolean
            }
        }
    }
}
id unique identifier for the project optional
description Updated description for the project optional
disabled boolean representing if the project is deleted optional
dashboard boolean representing if the project is in the dashboard optional

Response Syntax

        
    {
    "json": {
        "response": {
            "statusCode": "0",
            "statusMessage": "Success"
        }
    }
}

Error Codes

Status Code Description
0 Success
