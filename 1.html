SIGN UP

/functional/user/signup/ POST

This API is used to register a new user. A verification code will be sent to the registered user email id.

Request Syntax

        
    {
      "json": {
        "request": {
          "data": {
            "firstName": "String",
            "lastName": "String",
            "email": "String",
            "password": "String",
            "confirmPassword": "String",
            "mobile": "String",
            "countryCode": "String"
          }
        }
      }
    }
firstName The first name of the user mandatory
lastName The last name of the user mandatory
email User's email. This must be a work email mandatory
password User's password. Must be >8 characters mandatory
confirmPassword User's password entered again for confirmation mandatory

Response Syntax

        
    {
      "json": {
        "response": {
          "statusCode": "0",
          "statusMessage": "Success"
        }
      }
    }
statusCode Error code. 0 in case of no error.
statusMessage Error or Success message

Error Codes

Status Code Description
1003 Password doesn't match all constraints
1004 Password and confirm passwords don't match
1006 Password doesn't match all constraints
