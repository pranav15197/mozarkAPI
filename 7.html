Fetch devices

http://aquamark-dev.mozark.ai:8080/functional/devicegroup POST

List the devices available for testing. This list can be filtered based on model, manufacturer, os or the version of the device

Request Syntax

              {
  "json": {
    "request": {
      "session": {
      	"sessionId": "String"
      },
      "data": {
      	"os": "String",
      	"manufacturer": "String",
      	"version": "String"
      }
    }
  }
}

Request Parameters

The request accepts the following data in JSON format.

sessionId
Session Identifier
Type: String
Required: Yes
os
Field to be used to filter the devices based on the OS. Examples - ANDROID, IOS
Type: String
Required: No
manufacturer
Field to be used to filter the devices based on the manufacturer. Examples - apple, samsung
Type: String
Required: No
version
Field to be used to filter the devices based on the version.
Type: String
Required: No

Request Headers

The following headers must be sent with the request.

content-type
application/json
Required: Yes
