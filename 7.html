LIST DEVICES

/functional/devicegroup POST

List the devices available for testing. This list can be filtered based on model, manufacturer, os or the version of the device

Request Syntax

        
    {
  "json": {
    "request": {
      "session": {
      	"sessionId": "String"
      },
      "data": {
      	"os": "String",
      	"manufacturer": "String",
      	"version": "String"
      }
    }
  }
}
sessionId Session Identifier mandatory
os Field to be used to filter the devices based on the OS. Examples - ANDROID, IOS optional
manufacturer Field to be used to filter the devices based on the manufacturer. Examples - apple, samsung optional
version Field to be used to filter the devices based on the version. optional
