Create device group

http://aquamark-dev.mozark.ai:8080/functional/tray POST

This API is used to create a tray or a collection of devices that would further be used for automated testing.

Request Syntax

              {
    "json": {
        "request": {
            "session": {
                "sessionId": "String"
            },
            "data": {
                "description": "String",
                "list": [
                   "String",
                   "String"
                ]
            }
        }
    }
}

Request Parameters

The request accepts the following data in JSON format.

description
Description for the tray
Type: String
Required: Yes
list
A list of strings, where each string represents a device id that needs to be included in the tray
Type: String
Required: Yes

Request Headers

The following headers must be sent with the request.

content-type
application/json
Required: Yes

Response Syntax

              {
    "json": {
        "response": {
            "data": {
                "id": 3,
                "name": "TOPTAL_3"
            },
            "session": {
                "sessionId": "YGAAADAAAAAA"
            },
            "statusCode": "0",
            "statusMessage": "Success"
        }
    }
}

Response Elements

id
id of the created tray
Type: Integer
name
name of the created tray
Type: String
