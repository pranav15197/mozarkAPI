UPLOAD APK

/functional/appupload POST

This API is used to upload an APK to be tested. Please note that the call to this API will need to be through a multipart form-data request. The JSON request parameters shown below should be sent through a URL parameter named 'request'. The actual APK file has to be sent in form-data through a field named 'file'

Request Syntax

        
    {
  "json": {
    "request": {
      "session": {
      	"sessionId": "String"
      },
      "data": {
      	"projectId": Integer
      }
    }
  }
}
sessionId Session Identifier mandatory
projectId ID of the project to which this APK should be added mandatory
file The APK file mandatory

Response Syntax

        
    {
    "json": {
        "response": {
            "data": {
                "appId": 2
            },
            "session": {
                "sessionId": "YGAAADAAAAAA"
            },
            "statusCode": "0",
            "statusMessage": "Success"
        }
    }
}
appId A unique ID for the uploaded APK
