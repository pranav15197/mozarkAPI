MOZARK API DOCUMENTATION

List Scripts

/functional/script POST

Fetch all the scripts uploaded for a user

Request Syntax

              {
  "json": {
    "request": {
      "session": {
      	"sessionId": "{{session_id}}"
      }
    }
  }
}

Request Parameters

The request accepts the following data in JSON format.

sessionId
Session Identifier for a user
Type: String
Required: Yes

Response Syntax

              {
    "json": {
        "response": {
            "data": {
                "count": 1,
                "list": [
                    {
                        "id": 3,
                        "title": "Jobztop_Jobztop",
                        "language": "Java",
                        "version": "1.8",
                        "description": "JobzTop functional script",
                        "createdTime": "2020-04-19 06:18:26"
                    }
                ]
            },
            "session": {
                "sessionId": "YGAAADAAAAAA"
            },
            "statusCode": "0",
            "statusMessage": "Success"
        }
    }
}

Response Elements

count
Number of scripts returned
Type: String
List
List of objects with details for each script
Type: String
