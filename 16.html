Execute Tray

/functional/job POST

This API is used to execute a test script on an application for a tray (a user-selected group of devices)

Request Syntax

        
    {
    "json": {
        "request": {
            "session": {
                "sessionId": "String"
            },
            "data": {
                "appId": Integer,
                "scriptId": Integer,
                "trayId": Integer,
                "description": "String"
            }
        }
    }
appId ID for the application build on which the test should be run mandatory
scriptId ID for the test script that is to be run mandatory
trayId ID of the tray of devices to be used for testing mandatory

Response Syntax

        
    {
  "json": {
    "response": {
      "session": {
        "sessionId": "BBRAAXAAAAAA"
      },
      "statusCode": "0",
      "statusMessage": "Success"
    }
  }
}

Error Codes

Status Code Description
1009 The Tray ID provided doesn't exist
